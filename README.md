# Mudey's Catering Services

**Professional catering website** for Mudey's Catering Services in Accra, Ghana.

## ✨ Features

- **Elegant Design** - Modern, premium catering website with sophisticated color palette
- **Fully Responsive** - Mobile-optimized for all devices
- **Performance Optimized** - Fast loading with compressed images and optimized scripts
- **SEO Ready** - Meta tags, OG properties, and semantic HTML
- **Smooth Animations** - Scroll-triggered fade-ups using Intersection Observer
- **Contact Integration** - Phone call and email buttons

## 📱 Responsive Design

- Desktop (1200px+)
- Tablet (900px - 1200px)
- Mobile (600px - 900px)
- Small Mobile (<600px)
